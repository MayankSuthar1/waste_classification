About Waste Classification

üéØ Project Overview

This is an AI-powered waste classification system that uses deep learning and machine learning techniques to automatically categorize different types of waste materials. The system helps in proper waste segregation, which is crucial for effective recycling and environmental protection.

üß† Methodology

1. Image Preprocessing

Images are preprocessed using pixel intensity normalization, resizing to 224x224 pixels, and data augmentation techniques including rotation, flipping, and color jittering.

2. Feature Extraction with SqueezeNet

SqueezeNet is a lightweight convolutional neural network that extracts meaningful features from images. It uses Fire modules with squeeze and expand layers to achieve high accuracy with minimal parameters.

Key advantages:

  • 50x fewer parameters than AlexNet
  • Fast inference time
  • Suitable for deployment on resource-constrained devices

3. Classification with XGBoost

XGBoost (Extreme Gradient Boosting) is used as the final classifier. It takes the feature embeddings from SqueezeNet and performs multi-class classification.

Benefits:

  • High accuracy and fast training
  • Handles non-linear relationships well
  • Built-in feature importance analysis
  • Robust to overfitting

4. Evaluation

The model is evaluated using accuracy, precision, recall, F1-score, and confusion matrices to ensure reliable performance across all waste categories.

üìä Waste Categories

The system can classify the following waste types:

  • Cardboard - Recyclable paper-based material
  • Glass - Bottles, jars, and glass containers
  • Metal - Cans, foils, and metal objects
  • Paper - Documents, newspapers, magazines
  • Plastic - Bottles, bags, and plastic packaging
  • Trash - Non-recyclable waste

Note: Categories can be customized based on your dataset.

To train the model with your own dataset:

  2. Run python train_model.py
  3. Model will be saved in models/ folder
  4. Start the web app with python app.py
